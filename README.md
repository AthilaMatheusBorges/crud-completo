# crud-completo
Sistema de cadastro de jogos utilizando React no front e Node.js no back. 

Funcionalidades

- Criar jogos
- Ver os jogos
- Adicionar jogos 
- Deletar jogos
- Clicar no jogo para editar as informações ou editá-lo

Front-end
- React
- @material-ui
- Axios

Back-end
- Node
- express
- mysql
- nodemon
- cors
